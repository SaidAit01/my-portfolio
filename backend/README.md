cd to backend/StockTok/MarketDataService
Create and activate a virtual environment to use for project
1. python -m venv venv
2. source venv/bin/activate -> for MacOS/Linux
   venv\Scripts\activate.bat -> Windows
   Install the packages with requirements.txt file
   pip install -r requirements.txt
   Run FastAPI server
   uvicorn server:app --reload
